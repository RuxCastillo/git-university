# Contributing to git-university

Thank you for your interest in contributing!

## Contributor License Agreement (CLA)

Before your pull request can be merged, you must sign the
[Contributor License Agreement](CLA.md).

This is a **Copyright Assignment CLA**: it transfers the copyright of your
contribution to the project owner so the project can be maintained and
potentially licensed under additional terms in the future.

### How to sign

1. Open or comment on a Pull Request.
2. The CLA Assistant bot will automatically post a comment asking you to sign.
3. Reply with the following comment to confirm your signature:

   ```
   I have read the CLA Document and I hereby sign the CLA
   ```

4. The bot will record your signature and unblock the PR.

## Getting Started

1. Fork the repository.
2. Create a feature branch from `main`.
3. Make your changes.
4. Open a Pull Request against `main`.

## Code Style

- TypeScript is required for all new code.
- Run `npm run lint` and `npm run typecheck` before submitting.
